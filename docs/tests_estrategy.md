# Estratégia de Testes - Projeto Qualidade Rubeus

## Critérios para Automação vs Teste Manual

### O que AUTOMATIZAR ✅

1. **Testes de Regressão**
   - Verificação de elementos essenciais presentes
   - Fluxos críticos
   - Consistência de dados
   - Validação de links e botões

2. **Testes de Performance**
   - Tempo de carregamento
   - Responsividade básica

3. **Testes de Consistência**
   - Padronização de elementos
   - Verificação de conteúdo duplicado

### O que TESTAR MANUALMENTE

1. **Usabilidade e Experiência do Usuário**
   - Fluxos confusos
   - Elementos fora de contexto
   - Intuitividade da interface

2. **Testes Exploratórios**
   - Cenários não previstos
   - Comportamentos inesperados
   - Combinações de ações

3. **Design e Layout**
   - Alinhamento visual
   - Cores e contraste
   - Responsividade em dispositivos reais

4. **Conteúdo e Copywriting**
   - Textos "Lorem Ipsum"
   - Tom de voz e adequação
   - Erros de digitação e gramática
